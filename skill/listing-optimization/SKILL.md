# 跨境电商 Listing 优化

为跨境电商平台撰写高转化、符合平台规则的产品 Listing。

## 工作流程

1. **先确认信息**（缺失则主动询问）：产品类目、核心卖点、目标市场/语言、目标人群、品牌名、关键规格、价格区间、主要竞品。
2. 围绕 **买家搜索词 + 痛点 + 使用场景** 组织文案，而非罗列参数。
3. 输出结构见下。

## 输出结构

**标题（Title）**
- 公式参考：`品牌 + 核心关键词 + 关键属性/规格 + 适用场景/人群 + 差异化卖点`
- Amazon 标题 ≤ 200 字符；首词放最重要关键词；不堆砌、不全大写、不含促销词(如 "best/sale")。

**五点描述（Bullet Points）**，每条：
- 以**大写的利益点短语**开头，再展开说明（利益优先于功能）。
- 覆盖：核心功能、材质/工艺、使用场景、兼容/尺寸、售后/保障。

**产品描述（Description / A+）**
- 讲场景与故事，分段，可用要点；强调差异化与信任(认证、质保)。

**后台关键词（Search Terms）**
- 列出未在标题/五点中重复出现的长尾词、同义词、拼写变体；不含品牌侵权词、不重复、用空格分隔。

## 规则
- 严格遵守目标平台规则（如 Amazon 禁止促销语、保证、竞品对比、联系方式）。
- 目标语言**地道本地化**，不是直译；度量单位、用词习惯随市场调整。
- 每个卖点尽量给出**可信支撑**（数据、材质、认证），避免空洞形容词。

## 示例（节选）
> 标题：`AKAImage 加厚硅胶烘焙垫 40x30cm 不粘耐高温 烤箱专用 婴儿辅食可用`
> Bullet：`【告别粘黏与刷锅】食品级铂金硅胶，天然不粘，烘焙后一擦即净，省去反复刷烤盘的烦恼。`
