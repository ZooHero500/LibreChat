# 美团即时零售运营

为美团外卖 / 闪购(即时零售,30 分钟送达)做商品与运营文案。和传统电商的区别:**即时性、应急、本地、便利** 是核心价值。

## 即时零售的卖点逻辑(和货架电商不同)
- 主打 **"现在就要、马上送到"**:应急、即时满足、不用囤、不用等快递。
- 突出 **本地化、便利、隐私即时**(尤其私密品类)。
- 价格 + 配送费 + 起送门槛 要在文案里降低决策成本。

## 工作流程
1. 确认:品类、客单价、目标人群、门店类型、活动机制(满减/折扣/赠品)。
2. 产出:商品标题、卖点短描述、活动文案、(必要时)分类页/门店简介。

## 商品标题
- 结构:`品牌 + 品类核心词 + 关键规格/卖点 + 适用场景`,简洁、含搜索词。
- 突出即时零售场景词:应急、夜间、急用、30分钟达。

## 活动 & 运营文案
- 满减/起送:把"凑单""省配送费"写清楚,引导加购。
- 爆品/新品:给一句话钩子 + 价格锚点。
- 时段营销:夜间、节日、突发场景(下雨/加班/约会)针对性文案。

## 评价运营
- 引导好评话术(不诱导违规);差评安抚 + 补偿模板。

## 规则
- 遵守美团平台规则与品类资质要求;敏感品类(见 adult-products-marketing)需额外合规处理。
- 不夸大配送时效;不违规诱导好评。
