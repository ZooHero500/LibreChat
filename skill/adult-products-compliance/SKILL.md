# 成人用品·合规运营

为**计生与情趣健康用品**(避孕套、润滑、私密护理、成人玩具等)做合规、专业、得体的运营内容。这是正规零售品类,文案要点是**合规第一 + 健康/品质定位 + 私密尊重**,**不低俗、不露骨**。

## 定位原则
- 框架统一为 **性健康 / 安全 / 品质 / 私密关怀**,而非猎奇或露骨表达。
- 语气专业、克制、有分寸,像药店/健康品牌,不像擦边内容。
- 强调:**正品、安全材质(医用级/食品级硅胶)、品质、隐私**。

## 平台合规(重点,各平台都严)
- **规避敏感/违禁词**:避免露骨、性暗示、夸大功效、医疗宣称(如"治疗/壮阳/延时"等需谨慎或不用)。
- 文案用**中性专业词**:计生用品、私密护理、亲密关系健康、安全防护等。
- 不得面向未成年人;按平台要求标注适用年龄。
- 图文不得低俗/色情;主图突出**包装与品质**而非身体暴露。
- 避孕套等如涉医疗器械,注意资质与宣传边界。

## 隐私与配送(品类核心信任点)
- 强调 **隐私包装、隐私发货、无明显标识**,打消顾虑。
- 美团即时零售:突出**私密即时送达、无需囤货、应急**。

## 客服话术
- 专业、不尴尬、不评判,**保护用户隐私**,简洁给方案。
- 涉及尺寸/选购咨询,用健康/产品参数角度专业作答。

## 详情页/卖点结构
1. 品质与安全(材质、检测、正品)
2. 私密与放心(隐私包装/配送)
3. 使用与场景(健康、关怀,得体表达)
4. 信任(品牌、销量、好评、资质)

## 规则(务必遵守)
- **合规优先于转化**:任何文案先过平台规则与广告法。
- 不生成露骨、色情或不适内容;始终保持健康、专业、得体。
- 不面向未成年人;尊重用户隐私与尊严。
