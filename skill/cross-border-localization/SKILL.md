# 跨境多语言本地化文案

把中文（或英文）营销内容**本地化**到目标市场，让它读起来像母语者写的。

## 核心原则：本地化 ≠ 翻译
- 保留**意图和卖点**，重写表达，使其符合目标语言的地道说法与节奏。
- 调整**度量单位、货币、日期格式、尺码体系、电压/插头**等。
- 适配**文化与禁忌**：颜色寓意、宗教/节日、幽默与俚语、合规用语。

## 工作流程
1. 确认：目标市场/语言、内容类型(listing/广告/邮件/社媒)、品牌调性、是否有不可改的术语表。
2. 先理解原文意图 → 用目标语言重写 → 校对地道度与合规。
3. 输出：本地化文案 + （如有改动）**改动说明**(为什么这样改)。

## 各市场要点（参考）
- **美国**：直接、利益导向、口语化；避免夸大医疗/功效宣称。
- **德国/欧洲**：严谨、重数据与认证(CE)、注重隐私(GDPR)措辞。
- **日本**：礼貌、含蓄、重细节与品质感；敬语得当。
- **中东**：注意宗教与文化禁忌；阿拉伯语从右到左。
- **东南亚**：价格敏感、促销驱动、社媒/网红口吻。

## 规则
- 不确定的专有名词/品牌名**保留原文**并标注。
- 涉及功效、安全、比较级宣称时，按当地广告法**降级或加限定**。
- 需要时同时给出**多语言版本对照**。
